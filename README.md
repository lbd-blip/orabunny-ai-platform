# Orabunny AI Platform

## Overview
Orabunny AI is an AI-powered skin analysis platform that combines Vision AI and Generative AI to deliver personalized skincare routines.

## Core Technologies
- **Vision AI (LIQA)**: Live Image Quality Adjustment to correct lighting and angle distortions in user images.
- **Generative AI (RAG)**: Context-aware recommendation engine using retrieval-augmented generation.

## Key Features
- Skin condition analysis from mobile images
- Personalized skincare routine generation
- Continuous improvement through user feedback loops

## Project Status
This repository represents the early-stage technical structure of the Orabunny AI platform.  
Certain core algorithms are abstracted due to ongoing patent applications.
